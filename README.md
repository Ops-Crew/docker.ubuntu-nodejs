#### Setup ####

```shell
$ sudo docker pull tbaltrushaitis/ubuntu-nodejs
```
or with specific version of node e.g. 8.0.0
```shell
$ sudo docker pull tbaltrushaitis/ubuntu-nodejs:v8.0.0
```
